Problem
Assume array of integers. Give maximum sum of elements of a contiguous subarray.
A = [-6, 2, -4, 1, 3, -1, 5, -1]

Solution
Create a table containing arrays A and a new array B.
Initialise first element of B with max(A[1], 0):
